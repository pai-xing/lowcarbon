# 快速启动指南

## 前置要求

1. Node.js 16+ 和 npm/yarn
2. 后端服务已启动并运行在 `http://localhost:8080`

## 启动步骤

### 1. 安装依赖

```bash
cd frontend
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

前端将在 `http://localhost:3000` 启动

### 3. 访问应用

打开浏览器访问：`http://localhost:3000`

## 测试账号

### 普通用户
- 可以通过注册页面创建新账号
- 用户名：任意（3-50个字符）
- 密码：任意（6-20个字符）

### 管理员账号
需要在数据库中手动设置用户角色为 `admin`：

```sql
UPDATE tb_user SET role = 'admin' WHERE username = 'your_username';
```

## 常见问题

### 1. 无法连接后端API
- 确保后端服务已启动
- 检查后端端口是否为 8080
- 检查 `vite.config.js` 中的代理配置

### 2. 登录后无法访问管理员功能
- 确保用户角色在数据库中设置为 `admin`
- 重新登录以刷新用户信息

### 3. 页面样式异常
- 清除浏览器缓存
- 重新安装依赖：`rm -rf node_modules && npm install`

## 开发说明

- 修改代码后会自动热重载
- API 请求会自动添加 Token 认证
- 未登录访问受保护页面会自动跳转到登录页
